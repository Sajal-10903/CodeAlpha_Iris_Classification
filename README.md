# Iris Flower Classification using Machine Learning

## Project Overview

This project builds a supervised Machine Learning model to classify Iris flowers into three species based on their physical measurements.

The model predicts one of the following species:

- Iris Setosa
- Iris Versicolor
- Iris Virginica

This is a multi-class classification problem solved using the Random Forest algorithm.

---

## Dataset Information

The dataset contains 150 observations with 4 input features:

- Sepal Length (cm)
- Sepal Width (cm)
- Petal Length (cm)
- Petal Width (cm)

Target variable:

- Species (Setosa, Versicolor, Virginica)

Dataset source: Scikit-learn built-in Iris dataset

---

## Machine Learning Workflow

The following steps were performed:

1. Imported required libraries
2. Loaded and converted dataset into Pandas DataFrame
3. Performed exploratory data analysis (EDA)
4. Visualized feature relationships using Seaborn
5. Split dataset into training and testing sets
6. Trained Random Forest Classifier model
7. Evaluated model using accuracy, classification report, and confusion matrix
8. Saved trained model using Joblib

---

## Model Performance

The Random Forest Classifier achieved high accuracy (>95%) on the test dataset.

This demonstrates the effectiveness of ensemble learning for classification tasks.
